## OneAdvisor Client

### Getting started

-   `npm install`
-   `npm start`
-   Application should now be available at `https://localhost:5001`

### Testing

-   `npm run test`

### Linting

-   `npm run lint`
-   `npm run lint-fix`

### Webpack Bundle Analyzer

-   `npm run analyze:build`
-   `npm run analyze:run`

### Package version updates

-   `npm install npm-check -g`
-   `npm-check -u -E`
