# OneAdvisor Client

## Getting started

-   `npm install`
-   `npm start`

## Testing

-   `npm run test`

## Linting

-   `npm run lint`
-   `npm run lint-fix`

## Webpack Bundle Analyzer

-   `npm run analyze:build`
-   `npm run analyze:run`

## Ensuring latest versions

-   `npm install npm-check -g`
-   `npm-check -u -E`
